## What is working documentation

Working documentation is a set of drawings and specifications by which builders implement a design project. Without it, even the most beautiful project remains just a picture.

### What's included in documentation

**1. General data**
- Project cover
- Working drawings list
- General instructions and symbols

**2. Layout solutions**
- Furniture layout plan
- Dimensional references
- Room schedule

**3. Electrical and lighting**
- Outlet and wiring plan
- Lighting plan
- Control schemes

**4. Ceilings**
- Ceiling plan
- Sections and details
- Element specifications

**5. Floors**
- Floor plan
- Layout scheme
- Junction details

**6. Walls and finishes**
- Wall elevations
- Material markings
- Junction details

### Why documentation is needed

- **Accuracy** — builders know what and where to do
- **Savings** — fewer rework and errors
- **Control** — easy to verify compliance with the project
- **Specifications** — accurate material calculation
