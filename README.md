# INTERIOR360

This project runs locally as a single Astro dev server.

Notes:
- One local server, one port.
- No dashboards yet.
- No database connected.
- No extra backend services required for local work.

## Run project

Command:

```powershell
cd "E:\SITE INTERIOR360"
npm.cmd run dev -- --host 127.0.0.1 --port 4321
```

Batch file:

```powershell
.\start-local-site.cmd
```

Local URL:

```text
http://127.0.0.1:4321
```

## Build project

Command:

```powershell
cd "E:\SITE INTERIOR360"
npm.cmd run build
```

Batch file:

```powershell
.\build-site.cmd
```
